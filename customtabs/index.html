Custom Tabs

Необходимые файлы для подключения

Для корректной работы необходимо скачать jQuery библиотеку 3.3.1

  • jQuery

Иницилизация

Создаем блок для табов:

Общий блок для табов - class="exampleTab js-tabsblock"

Блок для мобильного разрешения - class="exampleTab__mobileSelect js-mobile-tab-link"

Ссылка для смены таба - class="js-tab-link exampleTab__tab -active", так же у ссылки есть data-href с классом привязанным к открывающиму блоку, class="-active" показывает что данный блок открыт

Таб айтем - class="exampleTab__item js-tab__elem1 js-tab", class="js-tab__elem1" именной индификатор для связывания с ссылкой.

Далее необходимо написать js код для иницилизации табов.
