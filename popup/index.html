popUp

Slick slider адаптивный слайдер для сайта. Можно управлять колличеством элементов для каждого необходимого разрешения.

Необходимые файлы для подключения

  • BlockUi
  • jQuery

Иницилизация

Создаем наш popUp:

<a href="#examplePopUp" class="js-show-popup"> кнопка открытия формы где a href="#examplePopUp" это id открывающейся формы.

<div id="examplePopUp" class="popUpForm hidden-block"> id - связка с кнопкой.

Далее необходимо написать js код для иницилизации плагина.

Расположение

Стили для всех форм должны лежать в pop_up.css, а скрипт должен лежать в pop_up.js.
